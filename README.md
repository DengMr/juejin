# juejin Command Line tools

## install

```bash
npm install -g juejin
```

## Documentions

juejin.im网站分类以及命令

|  模块  |  命令  |
|------- | -------|
|  Android  |  android  |
|  前端  |  fe  |
|  iOS  |  ios  |
|  后端  |  be  |
|  设计  |  de  |
|  产品  |  pm  |
|  工具资源  |  tools  |
|  阅读  |  read  |
|  人工智能  |  ai  |

查看最新文章或者最热文章
|  文章类别  |  命令  |
|------- | ------- |
|  最新文章  |  new  |
|  最热文章  |  hot  |

E.g:如果你想看到的是前端的最热内容
```bash
juejin hot fe
```
